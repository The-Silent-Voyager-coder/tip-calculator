# Tip Calculator with Bill Split

Quickly computes **tip and per-person share** of a bill.

## Features
- One-tap preset tip buttons (10, 15, 18, 20, 25%)
- Custom tip percentage
- Bill split across any number of people
- Shows tip, total, per-person total, tip per person

## Tech
- Pure HTML / CSS / JavaScript - no dependencies.

## Run
Open `index.html` in a browser or visit the live GitHub Pages site.
